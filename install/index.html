After you install (30 seconds)

  1. Add Skim from your store. Click the button for your browser above and confirm the permission prompt. Skim appears in your toolbar.
  2. Open any .md file. A markdown URL, a GitHub raw link, or a file on your machine. Skim renders it into a page automatically.
  3. Allow local files (optional). To read file:// markdown, click the Skim icon: it walks you through the single Allow access to file URLs switch and shows live when it's on. That's the only permission it ever asks for.

Prefer no install? The free online viewer renders any .md file in your browser, on any platform, nothing uploaded.
