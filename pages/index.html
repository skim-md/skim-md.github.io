Why Skim

Built for the AI era

Live reload

Keep an agent's plan.md open in a tab. Skim re-renders the moment the file changes on disk, scroll position preserved.

One-switch local files

Chrome blocks extensions from file:// pages by default. Skim walks you through the single toggle once, then never nags again.

Math, diagrams, tables

KaTeX math, mermaid diagrams, GitHub-flavored tables with smart handling for wide content. Hebrew and English bidi done right.

Folder view, free

Browse a folder of markdown files and hop between siblings. Other viewers charge for this. Skim never will.

Copy for AI

One click copies the document (or a selection) as clean markdown source, ready to paste back into a chat.

Private by design

No accounts, no telemetry, no network calls with your content. The store privacy declaration reads: no data collected.

Honest by policy
