Why Skim

Built for the AI era

Live reload

Keep an agent's plan.md open in a tab. Skim re-renders the instant the file changes, scroll position kept.

One-switch local files

Chrome blocks file:// by default. Skim flips the single switch with you, once, then never nags.

Math, diagrams, tables

KaTeX math, mermaid diagrams, GitHub-flavored tables. Hebrew and English bidi done right.

Folder view, free

Browse a folder of markdown and hop between siblings. Other viewers charge for this. Skim never will.

Copy for AI

One click copies the document, or a selection, as clean markdown source, ready to paste back into a chat.

Private by design

No accounts, no telemetry, no network calls with your content. The store privacy line reads: no data collected.

The real thing

A document, not a text dump

Every .md file opens with a contents sidebar that tracks where you are as you scroll, syntax-highlighted code, real tables, and light or dark themes. The same page you'd get on GitHub, for files that never leave your machine.
